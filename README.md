# マイ作物紹介サイト

西条農業高校の「農業と情報」実技テスト用に作成したWebサイトです。
育ててみたい作物「ナス」と「ミニトマト」についてまとめています。

## このサイトの工夫した点（4つの加点項目を達成）
1. **JavaScriptによる動的ギミック**：関連リンクにあるピンク色のボタンを押すと、応援アラートが表示され、ボタンの文字と背景色が変化します。
2. **2つの作物を紹介＆ページ内リンク**：上部にナビゲーションを設置し、ナスとミニトマトの紹介セクションへスムーズにスクロール移動できるようにしました。
3. **レスポンシブ対応**：スマートフォンやタブレットなどの小さな画面で見ても、画像や表、ボタンが画面からはみ出さず綺麗に折り返されるようにCSSを設計しました。
4. **READMEの作成**：本リポジトリにサイトの工夫点をまとめた本ファイルを同梱しました。
